Installing the PG Gem with PostgresApp

2015 / November 4

If you are having problem installing the pg gem and are using the Postgres app there is a good chance that the gem can’t find the correct configuration for Postgres since the config file is not availabe in $PATH.

Try installing the gem with the command below, remembering to change the version number to your installed version.

gem install pg -- --with-pg-config=/Applications/Postgres.app/Contents/Versions/9.4/bin/pg_config
